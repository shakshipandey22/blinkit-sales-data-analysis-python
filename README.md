# 🏪 Blinkit Sales Data Analysis (Python Project)

## 📌 Project Overview

This project performs a comprehensive end-to-end sales data analysis of Blinkit’s retail dataset using Python.

The objective is to evaluate:

- Sales performance  
- Product segmentation  
- Outlet performance  
- Location-based trends  

Using key KPIs and business-focused visualizations, this project uncovers actionable insights to support data-driven decision making.

---

## 🎯 Business Objectives

- Analyze the impact of fat content on total sales  
- Identify top-performing item types  
- Compare sales distribution across outlets  
- Evaluate effect of outlet establishment year on sales  
- Assess correlation between outlet size and revenue  
- Study geographic distribution of outlet sales  

---

## 📊 Key Performance Indicators (KPIs)

| KPI | Description |
|-----|------------|
| Total Sales | Overall revenue generated |
| Average Sales | Average revenue per item |
| Number of Items Sold | Total product count performance |
| Average Rating | Customer satisfaction indicator |

---

## 📈 Visualizations & Analysis

| No | Analysis | Chart Type |
|----|----------|------------|
| 1 | Total Sales by Fat Content | Bar / Donut Chart |
| 2 | Total Sales by Item Type | Bar Chart |
| 3 | Fat Content by Outlet | Stacked Column Chart |
| 4 | Sales by Outlet Establishment Year | Line Chart |
| 5 | Sales by Outlet Size | Pie / Donut Chart |
| 6 | Sales by Outlet Location | Bar Chart |

---

## 🔍 Key Insights

- Certain item types contribute significantly higher revenue.
- Outlet establishment year impacts total sales trends.
- Medium-sized outlets generate consistent revenue performance.
- Fat content segmentation shows measurable variation in sales behavior.
- Location-based differences reveal potential regional strategy insights.

## 🚀 How to Run This Project

### 1. Clone the repository

### 2. Navigate into the project folder

cd blinkit-sales-data-analysis-python

### 3. Install dependencies

pip install -r requirements.txt

### 4. Run Jupyter Notebook

jupyter notebook

Open the notebook file and run all cells.
